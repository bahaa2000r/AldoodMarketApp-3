# تعديلات الذمم والحركات المالية

تمت إضافة:
- شاشة الذمم / كشف الدين.
- إضافة دين عميل يدوي من الهاتف.
- تسديد دين عميل يدوي من الهاتف.
- تسديد دين فاتورة مبيعات أو مشتريات غير مدفوعة/جزئية.
- خانة المدفوع والمتبقي داخل فاتورة المبيعات والمشتريات.
- شاشة مدفوعات وسحوبات مالية.
- كل العمليات تحفظ محليًا داخل SQLite وتدخل ضمن ملف المزامنة `.adsync`.

ملاحظة:
- ديون الموردين اليدوية تكون عبر فاتورة مشتريات غير مدفوعة أو مدفوعة جزئيًا.
- الدين اليدوي المباشر مخصص للعملاء في هذه المرحلة.
