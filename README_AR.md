# الدود ماركت - Flutter Android Final Starter

هذه نسخة Flutter Android محسّنة تحتوي على:

- صفحة تسجيل دخول.
- الحسابات:
  - abd / 123
  - m / 123
  - d / 123
  - s / 123
- واجهة عربية RTL.
- ألوان داكنة مشابهة للبرنامج الحالي.
- شريط جانبي يمين.
- شعار المتجر.
- أزرار كبيرة.
- جداول بيانات.
- شاشات قابلة للتمرير.
- تصميم مناسب للموبايل والتابلت والكمبيوتر.
- شاشة إعدادات تلجرام.
- خدمة إرسال تلجرام عبر Bot Token و Chat ID.
- زر تجربة الإرسال.
- إرسال إشعار عند تسجيل الدخول إذا كانت إعدادات تلجرام مفعلة.

## تشغيل المشروع

بعد فك الضغط، افتح CMD أو PowerShell داخل المجلد ونفذ:

```powershell
flutter create --platforms=android .
flutter pub get
flutter run
```

## بناء APK

```powershell
flutter build apk --release
```

الملف الناتج غالبًا:

```text
build/app/outputs/flutter-apk/app-release.apk
```

## إعداد تلجرام

من داخل التطبيق:
1. افتح شاشة إعدادات تلجرام.
2. أدخل Bot Token.
3. أدخل Chat ID.
4. فعّل الإشعارات.
5. اضغط تجربة الإرسال.

بعد نجاح التجربة، التطبيق يصبح مربوطًا بتلجرام.
